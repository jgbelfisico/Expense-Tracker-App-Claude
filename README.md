# Expense Tracker

Aplicación web para registrar ingresos y gastos personales.

## Tecnologías

HTML, CSS y JavaScript puro — sin frameworks, sin backend, sin base de datos externa. Los datos se guardan en `localStorage`.

## Uso

Abre `index.html` en el navegador. No requiere servidor ni instalación.

## Funciones

- Registrar movimientos (ingreso o gasto) con categoría, monto, fecha y descripción
- Ver balance total, total de ingresos y total de gastos
- Filtrar movimientos por tipo y categoría
- Los datos persisten entre sesiones gracias a `localStorage`
